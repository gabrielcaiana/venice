# Breadcrumb

Breadcrumb allow users to make a deep navigation links.

To use it, you need to use `BreadcrumbItem` to each item of breadcrumb and add any HTML element as a children, for example `a` or `span`.

## Base

<Preview>
  <Story name="Base" parameters={{ decorators: [withA11y] }} inline={true}>
    {{
      components: { Breadcrumb, BreadcrumbItem },
      template: `
        <div style="padding: var(--space-sm)">
          <Breadcrumb>
            <BreadcrumbItem>
              <a href="/home" title="Home">
                Home
              </a>
            </BreadcrumbItem>
            <BreadcrumbItem>
              <a href="/item" title="Item">
                Item
              </a>
            </BreadcrumbItem>
            <BreadcrumbItem>
              <a href="/here" title="Here">
                Here
              </a>
            </BreadcrumbItem>
          </Breadcrumb>
        </div>
      `,
    }}
  </Story>
</Preview>
